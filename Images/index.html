CAD.io is a free online dicom viewing tool that aims to provide simple computer aided diagnostic tools.
According to The Application of Medical Artificial Intelligence Technology in Rural Areas of Developing Countries some countries have as few as one radiologist for every 2 million people. The goal of this project is to provide these radiologists with free tools to improve their workflow and increase the number of patient's they can treat. This website is also intended for radiology student to become familiar with computer aided diagnostics and has a variety of sample dicom images to practice with.
Unfortunately, at the moment, the dicom viewer does not have integrated computer aided diagnostics tools however a variety of diagnostic tools can downloaded from the github repositories linked in this website.
